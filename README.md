# Marvel_DMA

DMA-based external for Marvel Rivals.

## Setup

1. **Generate SDK** using [Dumper-7](https://github.com/Encryqed/Dumper-7?tab=readme-ov-file#how-to-use)
2. **Copy SDK contents** to `Marvel_DMA/Dependencies/Marvel Rivals SDK`
3. **Fix SDK errors related to UE5**
4. **Build** (x64)
